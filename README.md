# آمیرزا — بازیِ واژه‌سازیِ فارسی 🎯

یک بازیِ پازلِ کلماتِ فارسی شبیه به **آمیرزا**، ساخته‌شده با **React Native + Expo + TypeScript**.
بازیکن با چرخاندن و انتخابِ حروفِ روی چرخ، واژه‌های پنهانِ هر مرحله را می‌سازد، سکه جمع می‌کند و مرحله‌های بعد را باز می‌کند.

---

## ✨ امکانات

- 🔤 **چرخِ حروفِ دایره‌ای** با انتخابِ لمسی و خطِ اتصالِ بینِ حروف
- 🧩 **۵۰ مرحله‌ی آماده** با مجموعاً ۲۲۹ واژه (همگی اعتبارسنجی‌شده)
- 🪙 **سیستم سکه**: پاداش پس از هر مرحله + خرجِ سکه برای راهنما
- 💡 **سیستم راهنما**: نمایش یک حرف از کوتاه‌ترین واژه‌ی پیدانشده
- 💾 **ذخیره‌سازی محلی** با AsyncStorage (مرحله، سکه، مرحله‌های کامل، واژه‌های یافته)
- 🎵 **جلوه‌های صوتی** (ضربه، درست، غلط، برد، سکه) + **لرزشِ هپتیک**
- 🎬 **انیمیشن‌های موفقیت و خطا** (پالس و لرزش)
- 🧭 **ناوبری** با React Navigation (خانه، انتخاب مرحله، بازی، پیروزی، تنظیمات)
- 🌙 **رابط کاربری مدرن** با گرادیانِ بنفش و تمِ یکپارچه
- ↔️ **پشتیبانی کاملِ RTL** برای فارسی
- 📱 **واکنش‌گرا** برای اندروید و iOS
- 🧱 **معماری تمیز**: کامپوننت‌های قابل‌استفاده‌ی مجدد، هوک‌های سفارشی، Context API
- 🟦 **TypeScript** با حالتِ strict

## 🚀 راه‌اندازی

پیش‌نیازها: **Node.js 18+** و **npm**. (برای اجرا روی گوشی، اپ **Expo Go** را نصب کنید.)

```bash
# ۱. نصب وابستگی‌ها
npm install

# ۲. هم‌ترازکردنِ نسخه‌ی بسته‌ها با SDK (توصیه‌شده)
npx expo install --fix

# ۳. اجرا
npx expo start
```

سپس کدِ QR را با اپ **Expo Go** اسکن کنید، یا کلیدهای زیر را در ترمینال بزنید:

- `a` → اجرای اندروید
- `i` → اجرای iOS (روی macOS)
- `w` → اجرای وب

بررسی تایپ‌ها:

```bash
npm run typecheck
```

## 🎮 نحوه‌ی بازی

1. روی حروفِ چرخ ضربه بزنید تا یک واژه ساخته شود (حروفِ انتخابی بالای چرخ نمایش داده می‌شوند).
2. روی **بررسی واژه** بزنید. اگر واژه درست باشد، در خانه‌هایش سبز می‌شود.
3. تمام واژه‌های یک مرحله را پیدا کنید تا مرحله کامل شود و سکه‌ی پاداش بگیرید.
4. اگر گیر کردید، از **راهنما** استفاده کنید؛ یک حرف از کوتاه‌ترین واژه‌ی پیدانشده آشکار می‌شود (هزینه: ۳۰ سکه).
5. دکمه‌های کمکی: **بُر** (درهم‌کردنِ چرخ)، **حذف** (حذفِ آخرین حرف)، **پاک‌کردن** (پاک‌کردنِ انتخاب).

> نکته درباره‌ی تطبیقِ حروف: ورودی‌ها با یک مرحله‌ی **نرمال‌سازیِ فارسی** مقایسه می‌شوند
> (مثلاً «آ/أ/إ» → «ا»، «ي/ئ» → «ی»، «ك» → «ک»). به‌این‌ترتیب مثلاً با یک کاشیِ «ا» واژه‌ی «آب» ساخته می‌شود.

## 🛠 شخصی‌سازی

### افزودن مرحله‌ی جدید
در `src/data/levels.ts` یک شیء به آرایه‌ی `LEVELS` اضافه کنید:

```ts
{
  id: 51,
  letters: ['ک', 'ت', 'ا', 'ب'],   // حروفِ روی چرخ
  words: ['آب', 'تاب', 'کتاب'],     // واژه‌های صحیح
  reward: 30,
}
```

دقت کنید هر واژه باید فقط از حروفِ همان مرحله (پس از نرمال‌سازی) ساختنی باشد.

### تغییر صداها
فایل‌های WAV را در `assets/sounds/` با هم‌نام جایگزین کنید (`tap`, `correct`, `wrong`, `win`, `coin`).

### تغییر هزینه‌ی راهنما / سکه‌ی اولیه
در `src/constants/index.ts` مقادیرِ `HINT_COST` و `INITIAL_COINS` را ویرایش کنید.

### تغییر رنگ‌ها و ظاهر
همه‌چیز در `src/theme/index.ts` متمرکز است.

### افزودنِ فونتِ فارسی (مثلاً Vazirmatn)
1. بسته را نصب کنید: `npx expo install expo-font @expo-google-fonts/vazirmatn`
2. در `App.tsx` فونت را بارگذاری کنید (با `useFonts`).
3. در `src/theme/index.ts` مقادیرِ `fonts.regular` و `fonts.bold` را به نامِ فونت تنظیم کنید.

## 📦 درباره‌ی نسخه‌ی Expo

این پروژه روی **Expo SDK 54** (React Native 0.81 / React 19) تنظیم شده تا با **Expo Go**
سازگاریِ کامل داشته باشد و پایدار باشد. برای ارتقا به نسخه‌ی جدیدتر:

```bash
npx expo install expo@latest --fix
```

سپس راهنمای مهاجرتِ همان نسخه را در مستنداتِ Expo بررسی کنید. (پروژه از `expo-audio`
استفاده می‌کند؛ بسته‌ی قدیمیِ `expo-av` حذف شده است.)
